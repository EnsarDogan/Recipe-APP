Easy To Cook

Easy To Cook is a website that you can easily choose your menu for today and learn how to cook your meals

With this site you can get random meals, search a menu by name or filter meals by category and area!

You can search meal by name
Just write the name of meal that you wanna cook!

You can search meal by ingredient
Just write the name of main ingredient that you wanna use!

You can filter meal by category
Just write the meal category that you wanna explore!

You can filter meal by country
Just write the name of cousine that you wanna explore!
